WHAT THIS MODULE DOES:
======================
Takes a cropped license plate image (from YOLO detection) and straightens
it using top and bottom edge detection for horizontal alignment.

- Uses Canny edge detection
- Hough Line Transform to find horizontal edges
- Aligns plate based on detected edges
- Shows 3-panel visualization (Original | Straightened | Detected Edges)
